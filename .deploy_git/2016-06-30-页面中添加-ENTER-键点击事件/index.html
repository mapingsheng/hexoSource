页面中添加'ENTER'键点击事件

发表于 2016-06-30   |   分类于 javascript   |     |  

通常有以下场景：
当输入表单完成后直接按“Enter”键进行快捷提交；当操作一个共时，弹出的提示询问框，然后直接按“Enter”键确认。
那么我们只需要通过获取当前点击的键码是否是enter键的码值即可！

jquery

$("body").keydown(function(e){
    if(e.keyCode=="13"){
        alert('点击了Enter键');
    }
});

javascript

document.body.onclick=function(){
    if(e.keyCode=="13"){
       alert('点击了Enter键');
    }
};
